About Smart Library

This Smart Library Management System is a modern web application designed to simplify the process of managing books digitally.

Key Features:

  • Add and store books with images
  • Search books with suggestions
  • View detailed information in popup
  • Responsive design for mobile & desktop
  • Secure login system

This project is developed using HTML, CSS and JavaScript, and uses localStorage for data persistence.
